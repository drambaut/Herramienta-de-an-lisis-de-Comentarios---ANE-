# Herramienta-de-an-lisis-de-Comentarios---ANE-
Esta es la herramienta propuesta para el análisis de los comentarios que recibe la ANE, consiste en un pipeline que  realiza un análisis PLN, divide los comentarios por temas, crea gráficas relacionadas, y hace un análisis de IA para entender los comentarios e interpretar los resultados..
